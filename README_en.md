# LaserCut KonturFix

LaserCut KonturFix is a Windows app for preparing designs for laser cutter software.

The app helps make black-and-white stencil, papercut, and silhouette designs technically usable. Loaded images can be converted into a clean black-and-white mask, separated black areas can be connected automatically, and the result can be exported as PNG, SVG, or DXF.

## Features

- Load PNG/JPG/image files
- Create black-and-white masks
- Automatically connect separated black islands
- Set bridge width in millimeters
- Preview SVG contours
- Export optimized PNG
- Export SVG with red 0.2 mm cut lines
- Export DXF
- German and English Windows versions
