# LaserCut KonturFix

LaserCut KonturFix ist eine Windows-App zur Vorbereitung von Motiven fuer Lasercutter-Software.

Die App hilft dabei, schwarz-weisse Stencil-, Papercut- und Silhouettenmotive technisch besser nutzbar zu machen. Geladene Bilder koennen in eine klare Schwarz-Weiss-Maske umgewandelt, getrennte schwarze Bereiche automatisch verbunden und anschliessend als PNG, SVG oder DXF exportiert werden.

## Funktionen

- PNG/JPG/Bilddateien laden
- Schwarz-Weiss-Maske erstellen
- getrennte schwarze Inseln automatisch verbinden
- Brueckenbreite in Millimetern einstellen
- SVG-Vorschau anzeigen
- Export als optimiertes PNG
- Export als SVG mit roter 0,2-mm-Schnittlinie
- Export als DXF
- deutschsprachige und englischsprachige Windows-Version
