# LaserCut KonturFix v2.5.0

LaserCut KonturFix is a multilingual Windows tool for turning images and AI-generated motifs into laser-ready SVG/DXF files. Version 2.5.0 adds a dedicated engraving workflow with embedded raster engraving and a separate red cuttable outer contour.

## Highlights

- Create cut motifs from bitmap images
- Create engraving motifs with a cuttable outer frame
- Export SVG and DXF files for laser software
- Connect islands with adjustable bridges
- Adjust threshold, smoothing, cleanup and final motif width
- Multilingual interface: German, English, French, Italian and Spanish
- Saved language preference
- License workflow with installation ID and activation
- Installer with desktop shortcut and uninstaller
- Direct Custom GPT links for cut motifs and engraving motifs
